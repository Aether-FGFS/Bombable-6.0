                                        AI_MP_Bridge.
_____________________________________________________________________________________________

Overview : 

                 With AI_MP_Bridge, you can broadcast your bombable AI scenario via
                  multiplayer server. Clients will see your models flying around if AI
                  Scenario and models for AI Scenario are installed. Clients can destroy AI 
                  and will hear sounds and will get impacts, same as in singleplayer. Server 
                  clients without receiver will not see AI models.  



Installation :

                 Copy and paste folder Nasal under your FG root folder. Typicaly under 
                  user/AppData/Roaming/flightgear.org/Nasal. Install same AI models and
                  AI scenario as Server is broadcasting and you are ready to meet your
                  friends in combat situation around you. Install Scenarios into fgfs
                  installation folder fgdata/AI.

How to use : 


     Client:     Run FGFS, connect to same multiplayer server where AI_MP_Server
                  is running and enjoy AI models flying, driving, or sailing around you. 


     Server:     Run AI Scenario and connect to multiplayer server with MP callsign:Office
                  Use Bombable menu to bring models into your location. For mixed airplane/
                  ship Scenario, use both options to spawn them in air and ground around you.
                 

Dev.info :   
                 Create AI scenario with 8 models. Scenario Name must be max 29 letters
                  long. Only scenario <type>aircraft</type> and <type>ship</type> will be 
                  received at clients PC. Callsigns must be: Alpha,Bravo,Charlie,Delta,
                  Echo,Foxtrot,Golf,Hotel to be received. Receiver will auto load same
                  Scenario and models at same positions as Server PC is broadcasting.


PS:           
                 You can run Multiplayer mission with 9 bombable models include server model.
                  With updated models for faction , you can experience realistic missions,
                  where you are not middle of the universe anymore. Bombable 6.0.2 slider
                  setting for max number of AI attacking you, will be usefull to set your 
                  situation. Clients are still able to run their AI Scenario with static models
                  and objects. This way you can get large number of targets even for whole mp 
                  server with no performance impact.

                 Enjoy new features and have a good luck in Multiplayer missions.
